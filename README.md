# Cookie Clicker

## What's this project ?

> This project is a game tycoon-like made by a passionate developer in TypeScript.

## Installation

To install the dependencies, run the following command:

```bash
pnpm install
```

## Usage

To start the game, run the following command:

```bash
pnpm run dev
```
